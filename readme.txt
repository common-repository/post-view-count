== Description ==

By using this smart plugin, allows you to display how many times a post, page or custom post type had been viewed in a simple, fast and reliable way.

this 'view count' you can use for find more popuplar or most view your page, post or custom post and you can get analytic report for your website trrafic.  

**Plugin Features**

* Option to select post types for which post views will be counted and displayed.
* Dashboard post views stats widget
* Capability to query posts according to its views count
* Restricting display by user roles
* Restricting post views editing to admins
* Multisite compatibile
* W3 Cache/WP SuperCache compatible


== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

After active plugin go to Setting->Post View Counter and Select post type.
